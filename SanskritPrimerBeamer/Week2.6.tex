\subsection{形式与意义}
\begin{frame}{\insertsubsection }
  \begin{itemize}
    \item 形式层面：主动与中间语态语尾
    \item 意义层面：主动与被动含义
    \item 两者之间的关系：
    \begin{itemize}
      \item 主动语态表达主动含义。
      \item 现在时和未完成时有单独被动形式，被动语态使用中间语态语尾。
      \item 这两个时态的中间语态表主动含义。
      \item 其他时态中间语态表被动含义。
    \end{itemize}    
  \end{itemize}
\end{frame}
